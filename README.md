
# Things-with-spotify

Simple React app which will utilize spotify API.
Contains search bar to retrieve artists and display information such as:
 - most streamed songs
 - album names
 - trending songs
 - etc


## Requirements
ReactTs project

node v18.4.0

Material UI
## Deployment

To deploy this project navigate to /my-app and on the command line:

```bash
  npm start
```
